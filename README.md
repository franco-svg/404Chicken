# 404Chicken

Proyecto desarrollado como trabajo práctico, inspirado en el clásico juego del dinosaurio 404, implementado bajo el paradigma de **Programación Orientada a Objetos**.

## Descripción

404Chicken es un videojuego del género **endless runner**, donde el jugador controla una gallina que debe esquivar distintos obstáculos mientras el escenario avanza de forma continua.

El juego pone foco en la lógica de movimiento, colisiones, animaciones y control del flujo del juego, utilizando una estructura orientada a objetos que separa responsabilidades y facilita la extensibilidad del sistema.

404Chicken busca ofrecer una experiencia simple pero dinámica, priorizando la claridad del código, la organización de las clases y la correcta gestión de eventos dentro del ciclo del juego.

## Arquitectura

Aplicación desarrollada bajo un enfoque de **Programación Orientada a Objetos (POO)**, organizada en clases que representan las distintas entidades y comportamientos del juego.

## Tecnologías

- Python
- pygame (librería)

## Ejecución

Para ejecutar el juego localmente:

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/tuusuario/404Chicken.git
   ```
2. Acceder a la carpeta del proyecto
   cd 404Chicken
3. Instalar las dependencias necesarias (pygame en este caso)
   pip install pygame
4. Ejecutar y disfrutar del 404Chicken!!
   python main.py
